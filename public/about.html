About LK21 Vibe

Premium Movie Streaming Platform

Project Overview

LK21 Vibe adalah platform streaming film yang dibangun untuk tujuan edukasi. Project ini mendemonstrasikan penggunaan API, web scraping, dan modern web development.

Dibangun dengan HTML, CSS, dan JavaScript vanilla untuk frontend, serta Node.js/Express untuk backend API.

Tech Stack

HTML5
CSS3
JavaScript
Node.js
Express
Vercel

Features

  • Browse latest movies with real-time API
  • Filter movies by genre
  • Search functionality
  • Responsive design for all devices
  • Beautiful animations and transitions
