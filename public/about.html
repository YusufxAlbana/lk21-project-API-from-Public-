About LK21 Vibe

Project Overview

LK21 Vibe adalah platform streaming film yang dibangun untuk tujuan edukasi. Project ini mendemonstrasikan penggunaan API, web scraping, dan modern web development.

Dibangun dengan HTML, Tailwind CSS, dan JavaScript vanilla untuk frontend, serta Node.js/Express untuk backend API.
